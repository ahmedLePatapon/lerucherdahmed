# Step 2 — Configuration de l'internationalisation (i18n)

Commit: `feat: configure next-intl for FR/EN localization`

Objectif
--------
Configurer `next-intl` pour supporter le français (par défaut) et l'anglais, créer la structure de routes locale et fournir les messages de base.

Fichiers à ajouter
-------------------
- `middleware.ts` (détection et redirection de locale)
- `i18n.ts` (config locale centralisée)
- `messages/fr.json` (traductions françaises)
- `messages/en.json` (traductions anglaises)
- `app/[locale]/layout.tsx` (layout localisé)
- `app/[locale]/page.tsx` (page d'accueil localisée)
- modifier `next.config.ts` si nécessaire

Exemple de structure de `messages/*.json`
```json
{
  "nav": {
    "home": "Accueil",
    "honey": "Notre Miel",
    "about": "À propos",
    "contact": "Contact"
  },
  "common": {
    "order": "Commander"
  }
}
```

- Vérifier que `/` redirige vers `/fr`
